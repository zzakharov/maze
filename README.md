# Описание

Необходимо написать на php программу, которая будет выполнять следующее:
1. Загружать картинку с лабиринтом. Размер одной
клетки - 10x10 px;
2. Искать в лабиринте точки входа и выхода. Вход - красная точка (FF0000), а выход -
синяя (0000FF);
3. Искать наиболее оптимальный путь между этими точками;
4. Рисовать найденный в п.3 путь на загруженной картинке (ширина линии - 2px);
5. Выводить на страницу итоговую картинку с лабиринтом и найденным путем между
точками.

# Требования

Программа должна быть реализована с учетом следующего:
1. Нативный php 7.2+ и его расширения, без использования сторонних фреймворков
и библиотек;
2. Программа должна состоять из 1 скрипта для запуска программы и одного или
нескольких php классов (каждый класс в отдельном файле);
3. Программа не должна нуждаться в установке и должна работать в любом
окружении php7.2+, где есть расширения curl, ImageMagick, gd2.
